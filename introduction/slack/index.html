Slack简介

Slack 是目前北美主流的项目成员协作通信工具之一，将分散的沟通方式聚集到一个地方，将邮件，IM，短信，wiki等统一整合。具备聊天群组大规模工具集成文件统一搜索的特点，并支持桌面，WEB在线和移动版本，提供丰富的第三方应用集成，比如Facebook，Twitter，Github等。所以本次大数据黑客马拉松大赛，我们推荐此工具给大家，提供云服务器环境的即时IT技术支持，同样各参赛队也可以在上面创建自己的私有群讨论组，进行文件分享和即时通信。

来自大赛工作组的邀请邮件应该已经发送到你注册的邮箱，请根据提示，加入我们的China Hackathon组. 如果你还未收到此邮件,请发送邮件到imcloud@ca.ibm.com 咨询！或者也可以与现场的竞赛工作人员联系.

除了可以在浏览器中使用Slack, 我们还建议使用原生应用程序:

  • Mac OS
  • Windows
  • iOS
  • Android
