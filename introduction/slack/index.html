Hack Spark
  • Spark Hackathon

Slack 是目前北美主流的项目成员协作通信工具之一，此工具功能强大，支持桌面，WEB在线和移动版本，并提供丰富的第三方应用集成，比如Github等。所以Spark黑客马拉松大赛，我们将在次平台上提供云服务器环境的即时IT技术支持，同样各参赛队也可以利用此工具创建自己的私有讨论小组，进行队员间的即时通信和分享。

来自大赛工作组的邀请邮件应该已经发送到你注册的邮箱，请根据提示，加入我们的Slack工作组. 如果你还未收到此邮件,请发送邮件到leow@ca.ibm.com 咨询！或者也可以与现场的大赛工作人员联系.

你可以在浏览器中使用Slack, 但是我们还是建议使用原生应用程序:

  • Mac OS
  • Windows
  • iOS
  • Android
