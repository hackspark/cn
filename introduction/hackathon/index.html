大数据Spark Hackathon竞赛及介绍

Apache Spark作为目前大数据社区最流行的开源项目，受到大数据开发人员的高度关注。我们希望通过连续24小时的竞赛，充分激发开发者的创意和灵感，利用大数据来解决生活中具体的问题。我们的大数据专家将会亲自帮助和指导参赛人员，分享第一手的大数据开发经验。

奖品信息

  • 第一名的团队: 每位成员获得一部iPad Air2 : 直接获得“东升杯”全球创业大赛的复赛资格,有机会获得最后的冠军奖励100万人民币. 为团队的创业和创新提供专业的辅导和资源,项目优先进入中关村东升科技园孵化器,与风险投资提供对接和推荐的丰富机会和渠道

  • 第二名的团队: 每位成员获得一只Moto360手表或等值奖品

  • 分类奖若干(最具潜力奖，最佳展示奖等): Fitbit, 大数据培训课程代金卷等（暂定）

参赛规则

  • 开发者以个人身份注册报名，以团队参赛提交产品，每个团队限提交1件产品
  • 允许个人开发者现场组队，每个团队1~4人，1人最多参与1个团队
  • 产品可以使用提前准备的资源，所有代码开发必须在比赛现场完成
  • 鼓励利用公共API或者数据进行创新，但必须是原创的想法和设计
  • 能够有效地利用大数据Spark的特性进行开发
  • 必须使用Data Scientist Workbench作为主要的开发工具和最终的演示平台
  • 每个参赛队将有10分钟的介绍和展示时间

产品提交

  • 每个参赛队的产品代码需在截至时间前提交到 GitHub 本次大赛的源码库
  • 提交产品至少包括：源代码，文字介绍和产品截图，还可以包括展示PPT等其他资料

现场规则

  • 允许在不影响他人开发的前提下自由交流
  • 禁止吸烟
  • 禁止大音量播放音乐等

评判标准

  • 创 意: 你的解决方案是否有新意，具有创新性?
  • 实践性: 你的解决方案是否能解决现实生活中得问题?
  • 功 能: 你的代码是否能够正常工作，演示是否流畅?
